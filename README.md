# Obsidian_uni

A ready-to-use Obsidian vault and templates tailored for university coursework, note-taking, and research. This repository provides an organized folder structure, note templates, and recommended plugin and workflow suggestions to help students and researchers manage lecture notes, readings, projects, and study plans.

## Features

- Organized folder structure for lectures, readings, projects, and daily notes
- Reusable note templates (lecture, reading, project, meeting)
- Suggestions for Obsidian community plugins and settings
- Example workflow for linking notes, spaced repetition, and revision

## Getting started

Prerequisites:
- Obsidian (https://obsidian.md)

Steps:
1. Clone or download this repository:
   ```bash
   git clone https://github.com/learner20011-gif/Obsidian_uni.git
   ```
2. Open Obsidian and choose "Open folder as vault" and select the cloned repository folder.
3. Install recommended community plugins .
